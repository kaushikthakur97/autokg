# Go-to-market package

## Core pitch

Turn warehouse/lakehouse tables into a governed knowledge graph for AI agents. autokg gives agents explicit entity relationships through MCP, with lineage, PII masking, and audit trails.

## Best wedge

Agent-ready semantic layer for structured enterprise data.

## First ICPs

1. Data platform teams building AI-agent infrastructure
2. Insurance/fintech teams with relationship-heavy regulated data
3. dbt/Databricks teams that need governed agent access to curated tables

## Paid pilot

**2-week Agent-Ready Knowledge Graph Pilot**

Deliverables:

- 5-10 source tables onboarded
- accountable relationship config
- MCP endpoint
- one executive demo question flow
- governance report
- production rollout plan
