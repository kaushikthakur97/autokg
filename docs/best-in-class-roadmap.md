# Best-in-class readiness

This checklist tracks the product areas needed for best-in-class status.

## Included now

- deterministic v1 graph compiler
- formal JSON Schema export
- strict config/data/FK validation
- ontology and SHACL generation
- RDF/JSON-LD/N-Triples/RDF/XML outputs
- manifest, lineage, audit, validation report, build report
- query backend with NL→SPARQL, SPARQL validation, execution, evidence
- LLM provider abstraction: mock, OpenAI, Anthropic, Gemini, Ollama, custom HTTP
- REST API with OpenAPI document
- MCP tools for schema, SPARQL, NL ask, sessions, manifest, lineage
- query audit JSONL and metrics registry
- eval runner for NL→SPARQL and answer checks
- benchmark command
- graph store abstraction for RDFLib and remote SPARQL

## Still naturally iterative

- deeper semantic planner and entity linker
- enterprise RBAC/ABAC UI
- distributed graph builds for very large data
- managed cloud/control plane
- richer web Studio
