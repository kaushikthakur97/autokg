# autokg Roadmap

## v0.4 — Sellable developer experience
- Declarative `autokg.yml` builds
- `autokg init` starters for Customer 360 and Insurance
- Docker and Docker Compose
- Static `autokg studio` dashboard
- Claude Desktop and Cursor MCP guides

## v0.5 — Modern data stack integrations
- dbt manifest connector
- Databricks notebook quickstart
- Snowflake connector guide
- DataHub/OpenMetadata export
- Airflow/Dagster examples

## v0.6 — Enterprise MCP gateway
- OAuth/OIDC for HTTP MCP
- RBAC and read-only policy profiles
- Query allowlists, max result rows, timeouts
- Prometheus/OpenTelemetry metrics
- Agent query audit dashboard

## v1.0 — Production semantic layer
- Stable API and config schema
- Helm chart and Terraform examples
- Large-scale benchmarks: 1M, 10M, 100M rows
- SOC 2 readiness documentation
- Enterprise support package
